Documentation Page

This myFlix API is one of the server-side component of the myFlix web application.

This API was built using JavaScript and Node.js. It interacts with a database that stores data about different movies, and will be accessible via the HTTP methods GET and POST.

Business Logic URL HTTP Request Method Request body data format Response body data format
Return a list of all movies /movies GET None JSON object holding data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by Name /movies/:title GET None JSON object holding data about a single movie containing properties for a description/genre/director/image URL
Return data about a genre (description) by Name (e.g., “Thriller”) /genres/:name GET None JSON object holding data about a specific genre containing a description with data
Return data about a director (bio, birth year, death year) by name /directors/:name GET None JSON object holding data about a specific director containing a description with data
Allow new users to register /users POST JSON object holding data about user to be added JSON object holding data about new user that was created
Get user by username /users GET JSON object holding data about user you're looking for JSON object holding data about specific user you searched for by Username
Allow users to add a movie to their list of favorites /users/favorites/add/:title POST JSON object holding data about movie to be added to favorites Show only a text that a movie has been added
Allow users to update their user info (username) /users/update/:username PUT JSON object holding data about user to be updated JSON object holding data about user that was updated and text indicating whether the user was successfully updated
Allow existing users to deregister /users/deregister/:username DELETE JSON object holding data about user to be removed Text indicating whether user was successfully removed
Allow users to remove a movie to their list of favorites /users/favorites/remove/:title DELETE JSON object holding data about movie to be removed from favorites Show only a text that a movie has been removed
